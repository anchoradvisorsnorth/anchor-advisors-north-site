# Asset Export Notes

SVG source files are included for all requested logo variants and OG image.

## Included
- `logo-primary.svg`
- `logo-stacked.svg`
- `logo-white.svg`
- `favicon.svg`
- `og-image.svg`

## PNG/Web-Optimized Export
Export PNG variants from these SVGs in your design tool (Figma/Illustrator) or via CI tooling.
Recommended outputs:
- Logo PNGs: 2x transparent background
- Favicon PNG: 32x32 and 64x64
- OG image PNG: 1200x630 (`og-image.png`)

After exporting, place files in this same `assets/` directory and (optional) update OG image meta tag to use PNG.
